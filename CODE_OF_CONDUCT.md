# Code of Conduct

## Our Standard

We want `YT Notes` to be a friendly, low-friction project to contribute to.

Participants in this project are expected to:

- be respectful and constructive
- assume good intent and ask clarifying questions before escalating
- give feedback about code and ideas, not people
- welcome contributors with different levels of experience

Unacceptable behavior includes harassment, discrimination, personal attacks, trolling, or any sustained disruptive behavior.

## Scope

This code of conduct applies in the repository, issues, pull requests, and other project spaces.

## Reporting

If you experience or witness behavior that violates this code of conduct, open a private report with the repository maintainer if possible, or file an issue asking for maintainer contact.

## Enforcement

Project maintainers may remove comments, close discussions, or block participation when behavior makes the project less safe or less constructive.
